# Standard MSc Thesis Report

## Suggested Structure
1. Abstract
2. Introduction
3. Problem Statement
4. Related Work
5. Methodology
6. System Design
7. Implementation
8. Evaluation
9. Governance, Ethics, and Risk
10. Discussion and Trade-offs
11. Conclusion and Future Work

## Required Why Documentation
For each major technology or method include:
- Choice
- Alternatives considered
- Rationale
- Trade-offs
- Reconsideration conditions

## Appendices
- Command logs
- Phase and batch reports
- Validation evidence
