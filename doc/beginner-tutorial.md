# Beginner Tutorial

## Audience
New contributors onboarding to governed persona intelligence implementation.

## Learning Outcomes
- Understand the project architecture.
- Run local validation commands.
- Follow batch protocol and documentation standards.

## Quick Start
1. Clone repository.
2. Open in Codespaces or dev container.
3. Read roadmap and execution tracker.
4. Run baseline checks.

## First Contribution Path
1. Pick one batch scope.
2. Announce scope and objective.
3. Request approval.
4. Implement atomic change.
5. Update command logs and batch report.

## Success Criteria
- Contributor can execute one batch end to end with policy compliance.
